intellij-javadocs2
============

Intellij plugin Javadocs 中文增强版
因为原有插件很久不维护了，已经不适应新版本Intellij idea，所以对插件进行了修改，并进行了汉化。
同时增加了增强功能，比如支持接口元素

使用教程 https://github.com/freedommedal/idea-javadocs2/wiki

Intellij plugin Javadocs the Chinese language enhanced version
